# HTML evidence belongs to onboarding attempts

The Bancada de Refinamento de Extractors needs to explain and adjust persisted Extractors against the same Evidencia HTML that justified them. We store that evidence as historical snapshots attached to `agency_onboarding_attempts`, not directly on the Agency, so each onboard or reonboard keeps its own proof set and later site changes do not rewrite why a given Extractor was accepted.
